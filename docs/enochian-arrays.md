# Enochian Chess Starting Arrays

This document describes the eight starting arrays used in Enochian Chess as described by Chris Zalewski and the Golden Dawn papers.

## Overview

There are four Elemental Boards (Tablets):
1.  **Tablet of Fire**
2.  **Tablet of Water**
3.  **Tablet of Air**
4.  **Tablet of Earth**

Each Tablet can be set up in different orientations or "angles".
